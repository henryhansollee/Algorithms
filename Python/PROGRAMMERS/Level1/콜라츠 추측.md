# 콜라츠 추측

```python
def solution(num):
    answer = 0
    
    k = 0
    while True:
        if num == 1:
            answer = k
            break
        if k > 500:
            answer = -1
            break
        if num % 2 == 0:
            num /= 2
        else:
            num *= 3
            num += 1
        k += 1

    return answer
```
